# rpi5-ai-inference-llm-optimized-linux-kernel
A high-performance, headless Linux kernel for the Raspberry Pi 5, purpose-built for edge AI inferencing and local LLMs. It maximizes memory bandwidth using 16K pages, Transparent HugePages, and Fake NUMA emulation, while minimizing idle power draw with a 100Hz tickless design. Stripped of GUI and audio bloat to dedicate all RAM to your models.
